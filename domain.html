Research Problem

A collection of nodes which have the ability to move randomly within a wireless network is called a mobile ad hoc network. Mobile Ad Hoc Network plays a major role in wireless communication technology. Ensuring security in a mobile ad hoc network is a major concern due to the unpredictable motions and behaviors of network nodes.



Further Discussion..

In a wireless mobile network, it is possible for a large number of data packets to transmit among nodes within a small period of time. Therefore, it is possible that some nodes might not behave as expected. It can eventually cause to a considerable amount of data packet drops. It shows that the existing security mechanisms have failed to distinguish between trustworthy and malicious nodes. Usually, the nodes select the shortest path. But sometimes it may not be the reliable route to transfer data. Proposing a method to evaluate the trustworthiness of each node, will address the issue up to a reasonable extent.


Objectives :

  • Update trust table dynamically
  • Neighbor node recommendations
  • Identify and isolate malicious nodes from the network
  • Trust level classification
  • Most trustworthy route instead of the shortest path




Research Gap and Literature Survey

  AODV is a routing protocol designed for wireless and mobile ad hoc networks which are widely used in the global and the local market. The new trust protocol which is being implemented as an extension to AODV, with new features and functions to guarantee the security and reliability of a MANET. Instead of the shortest path method of traditional AODV, this protocol will consider the most secured path for forwarding packets from a source node to a destination node. This trust based routing protocol increases the performance and the security of MANET networks.

  Because of the challenges and the significance of the routing protocols, mobile ad hoc networks have become one of the most active research areas. There has been a number of research papers that have shown keen interests in this field.

  • Protocol based scheme [R. Dalal, Manju Khari] [1]
      Communication among network nodes happened through agents called ants.

  • Intrusion Detection Model (IDM), Intrusion Response Model (IRM) [T. Farid, A. Prahladachar] [2]
      Uses neighbor node information to identify and isolate malicious nodes.

  • ST-AODV [Sridhar Subramanian and Baskaran Ramachandran] [3]
      Calculate request rate, reply rate and data transmission rate of network nodes.

  • EBoX [Santhosh J and Malini V K] [4]
      Uses a reputation-based schema to award penalty points and credit points.
