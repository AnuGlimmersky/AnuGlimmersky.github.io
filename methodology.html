  • Data Packets calculation
  • Generate the trust table values
  • Store Direct Trust for neighbor nodes

Indirect Trust Calculation

  • Maturity model
  • Recommendation model
  • Weight factor for indirect trust calculation
  • Reduction factor

    Trust Recommendation Request Protocol

  • Request recommendation values from neighbor nodes
  • Recieve recommendations from neighbor nodes
